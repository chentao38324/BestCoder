<h2 id="2">2.两字符串旋转</h2>  
>如果对于一个字符串A，将A的前面任意一部分挪到后边去形成的字符串称为A的旋转词。  
比如A="12345",A的旋转词有"12345","23451","34512","45123"和"51234"。  
对于两个字符串A和B，请判断A和B是否互为旋转词。  

<h2 id="3">3.替换空格</h2>  
>把字符串中的每个空格替换成"%20"。因为如果URL中含有特殊字符  
可能导致服务器端无法获得正确的参数值。  
思路：从后向前遍历 时间复杂度O（n）  

<h2 id="4">4.排序数组合并</h2>  
>两个排序数组A1和A2,将A2插入A1仍然有序，同3从后向前遍历可实现O(n)复杂度
